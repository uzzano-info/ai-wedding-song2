# AI Wedding Song 2 - Project Rules

## Tech Stack
- HTML5, CSS3, Vanilla JS
- Mobile-first responsive design
- Vite for module bundling

## Styling & Aesthetic Guidelines
1. **Premium Elegance**: The design must feel high-end, tailored for wedding services. Use soft, romantic color palettes (warm whites, blush, soft gold) and elegant typography (e.g., serif for headings, clean sans-serif for body).
2. **Dynamic UI**: Use subtle, smooth micro-animations (fade-ins, soft scaling) to create a lively and polished experience.
3. **No TailwindCSS**: Vanilla CSS only, using CSS Variables for consistent theming and maintaining the project's custom aesthetic.
4. **Responsive Integrity**: Ensure perfect rendering across mobile and desktop. Mobile view is priority given typical wedding service browsing habits.

## Code Quality
- Keep code modular (e.g., `src/styles`, `src/components`).
- Ensure semantic HTML for SEO and accessibility.
- Comment complex JavaScript logic, especially audio players and animations.
