# AI Wedding Song 2 - Project Skills

## Custom Wedding Music Landing Page Development
This project requires specialized skills in building high-conversion, premium service landing pages without heavy frameworks.

### Key Competencies
1. **Vanilla JS Audio Integration**: Building custom audio players to showcase "Sample" vs "AI Random" song drafts without relying on generic HTML `<audio>` aesthetics.
2. **Pricing UI Modeling**: Translating the 3-tier (Standard, Deluxe, Premium) package into an intuitive, visually distinct pricing table.
3. **Copywriting Presentation**: Layout text to highlight emotional resonance (Love Story -> Custom Lyrics -> Song) rather than just technical features.
4. **Vite Build Optimization**: Managing static assets (images, audio snippets) efficiently for fast load times.
