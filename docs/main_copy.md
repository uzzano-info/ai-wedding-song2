## 3. 솔루션 & 전문성 (Solution Section)

| 요소 | 카피 |
| :--- | :--- |
| **영문 라벨** | PROFESSIONAL QUALITY |
| **섹션 타이틀** | 최첨단 기술에 프로듀서의 섬세한 디테일을 더하다 |

### 본문 카피
> 아무리 뛰어난 장비나 재료가 있어도 모두가 일류 셰프가 될 수 없듯, AI 기술 역시 '누가, 어떻게' 디렉팅하느냐가 결과물을 좌우합니다.  
> 전문가의 믹싱과 마스터링을 통과한 **실제 발매 수준의 프로 퀄리티 음원**만을 약속드립니다.  
> 단 하나뿐인 특별한 날을 위한 음악이기에 대충 만들지 않습니다. 두 분의 마음에 닿을 때까지 무드를 조율해 드립니다.

### 핵심 강점 카드 (3-Column)

| # | 타이틀 | 설명 |
| :--- | :--- | :--- |
| 1 | **인간의 터치** | 기계적인 산출물이 아닌, 작곡가/디렉터의 귀로 직접 다듬은 음악 |
| 2 | **스튜디오 퀄리티** | 스트리밍 플랫폼 수준의 고음질 마스터링 (WAV / MP3) |
| 3 | **다양한 활용도** | 본식 행진곡부터 모바일 청첩장 추억의 BGM까지 |

### 카피 의도
- "셰프 비유" → AI에 대한 품질 불안감을 전문가의 손길로 해소
- "실제 발매 수준" → 아마추어 느낌이 아닌 프로급 결과물 보장
- 3개 핵심 강점이 각각 '감성·품질·실용성'을 커버하여 구매 장벽 제거

## 4. 압도적인 차별화 (Differentiation Section)

| 요소 | 카피 |
| :--- | :--- |
| **영문 라벨** | DIFFERENTIATION |
| **섹션 타이틀** | 1차 초안 2곡 동시 제공: 확실한 퀄리티와 기분 좋은 서프라이즈 |

### 본문 카피
> 고객님의 사연을 바탕으로 완전히 다른 매력을 가진 2가지 무드의 초안을 먼저 들려드립니다.  
> 직접 비교해 보시고 우리의 이야기와 가장 잘 어울리는 곡을 선택하세요.

### 트랙 구성

| 트랙 | 이름 | 설명 |
| :--- | :--- | :--- |
| **Track A** | 검증된 시그니처 무드 | 기존 고객들에게 가장 사랑받았던, 퀄리티가 확실하게 보장되는 검증된 프리셋 무드 |
| **Track B** | 맞춤 AI 무드 | 설문 데이터를 분석하여 새롭게 창조하는 랜덤 무드. 예측 불가의 신선함을 당사 프로듀서진이 다듬어 완성합니다. |

### 사이드 패널 카피
- **타이틀**: 실패 없는 완벽한 선택
- **설명**: 안정적인 시그니처 트랙과, 우리만을 위한 AI 맞춤 창작 트랙. 두 가지의 다른 매력을 직접 경험해보세요.

### 카피 의도
- "2곡 동시 제공" → 경쟁사 대비 유일한 구조적 차별점을 섹션 타이틀로 바로 강조
- Track A "검증된" vs Track B "새롭게 창조" → 안정감과 모험심 두 가지 심리를 모두 충족
- 사이드 패널의 "실패 없는" → 구매 리스크를 정면으로 해소하는 안심 문구

> *패키지에 따라 제공 방식 및 선택 가능한 곡 수가 상이합니다.

## 6. 작업 프로세스 (Work Process)

| 요소 | 카피 |
| :--- | :--- |
| **영문 라벨** | HOW IT WORKS |
| **섹션 타이틀** | 쉽고 간편한 작업 순서 |
| **도입 문구** | 어렵게 생각하실 필요 없습니다. 작사, 작곡은 저희가 알아서 하겠습니다. 객관식 위주의 설문지만 가볍게 작성해 주세요. |

### 스텝 구성

| STEP | 타이틀 | 설명 |
| :---: | :--- | :--- |
| **1** | 사연 형태의 가벼운 설문 | 첫인상, 장소, 원하시는 템포 등 체크리스트 위주의 편한 질문지 작성 |
| **2** | 가사 컨펌 & 2개 데모 초안 | 전문가가 다듬은 가사를 확인하고, 전혀 다른 2가지 무드의 데모곡 중 하나를 셀렉 |
| **3** | 최종 음원 믹싱/마스터링 전달 | 극장/식장에서 바로 틀어도 손색이 없는 프로 수준의 스튜디오 음원 제공 (약 3일 소요) |

### 카피 의도
- 3스텝 구조 → "이것만 하면 된다"는 심리적 허들 최소화
- STEP 1에서 "체크리스트 위주"를 강조 → 작문 부담 제거
- STEP 3에서 "약 3일" → 구체적인 일정 제시로 고객 불안 해소

## 7. 패키지 안내 (Pricing Section)

| 요소 | 카피 |
| :--- | :--- |
| **영문 라벨** | PRICING |
| **섹션 타이틀** | 합리적인 패키지 구성 |

### 패키지 비교표

| 구분 | STANDARD (베이직) | **DELUXE (시그니처 / BEST)** | PREMIUM (콜렉터) |
| :--- | :--- | :--- | :--- |
| **가격** | 30,000원 | **55,000원** | 79,000원 |
| **한 줄 설명** | 빠르고 간편하게 완성하는 플랜 | **2가지 분위기를 직접 비교해보는 가장 확실한 선택** | 2가지 다른 컨셉의 음원을 모두 소장 |
| **추천 대상** | 심플하게 한 곡만 필요한 분 | **무드를 비교하고 최적의 곡을 고르고 싶은 분** | 본식 + 청첩장 등 용도별 2곡이 필요한 분 |
| **곡 구성** | 지정곡 1개 (시그니처 무드) | **지정곡 1개 + 맞춤 AI곡 1개** | 초안 무드 2가지 모두 제공 |
| **선택 방식** | 초안 1개 제공 | **초안 2개 중 1곡 최종 선택** | 최종 마스터링 2곡 모두 지급 |
| **가사 수정** | 1회 제공 | **2회 제공 (가장 안정적)** | 5회 제공 |
| **결과물** | 완곡 1곡 (MP3, WAV) | **완곡 1곡 (MP3, WAV)** | 완곡 2곡 (MP3, WAV) |

### 패키지별 카피 메모

- **STANDARD** — "사연 기반 커스텀 작사 + 지정곡 1개 + 완곡 1곡 제공 + 가사 수정 1회". 핵심 경험을 가성비 있게 누릴 수 있는 입문 패키지.
- **DELUXE** — "사연 기반 커스텀 작사 + 지정곡 1개 + 맞춤 AI곡 1개 + 총 2가지 무드 초안 비교 후 1곡 최종 선택 + 완곡 1곡 제공 + 가사 수정 2회". 가장 많은 고객이 선택하는 이유: 두 가지 무드를 직접 들어보고 결정할 수 있어 **후회 없는 선택**이 가능.
- **PREMIUM** — "사연 기반 커스텀 작사 + 초안 무드 2가지 모두 최종 마스터링 + 완곡 2곡 제공 + 가사 수정 5회". 본식과 리셉션, 또는 본식과 모바일 청첩장 등 용도별로 각각 다른 무드의 곡을 활용하고 싶은 분들을 위한 풀 패키지.

### 카피 의도
- DELUXE에 "BEST CHOICE" 뱃지 + "가장 확실한 선택" → 앵커링 효과로 중간 패키지 전환 유도
- "추천 대상" 행 신설 → 고객이 자기에게 맞는 패키지를 빠르게 식별
- PREMIUM의 "소장" 키워드 → 음원을 '소비'가 아닌 '컬렉션'으로 포지셔닝

## 8. 자주 묻는 질문 (FAQ)

### Q1. 만들어진 곡이 마음에 들지 않으면 어떡하나요?
> 사전에 설문지를 통해 원하시는 방향을 꼼꼼하게 파악하기 때문에 대부분 1차 초안에서 만족하십니다. 하지만 혹시라도 방향성이 맞지 않는다면, 제공된 시안 선택 및 수정 단계를 통해 최대한 원하시는 무드로 조율해 드립니다. 저희의 목표는 두 분의 완벽한 만족입니다.

### Q2. 가사를 직접 써야 하나요?
> 아닙니다! 두 분의 첫인상, 의미 있는 장소 등 간단한 객관식/단답형 설문만 작성해 주시면 전문 작사가의 감각으로 자연스러운 가사를 완성해 드립니다. 부담 없이 저희에게 맡겨주세요.

### Q3. 제작 기간은 얼마나 걸리나요?
> 설문 작성 완료 후 **가사 초안은 약 1일, 최종 음원 전달까지 약 3일** 정도 소요됩니다. 결혼식 일정에 충분히 맞출 수 있도록, 최소 1주일 전 주문을 권장드립니다.

### Q4. 어떤 장르나 스타일로 만들 수 있나요?
> K-발라드, 인디 팝, 어쿠스틱, 트로트, R&B, 크리스마스 무드 등 **다양한 장르에 대응 가능**합니다. 설문 시 원하시는 레퍼런스 곡이나 분위기를 알려주시면 최대한 반영해 드립니다.

### Q5. 결혼식 외에 다른 용도로도 사용 가능한가요?
> 물론입니다! 모바일 청첩장 BGM, 리셉션 음악, 프로포즈 영상 배경음악, 기념일 선물 등 다양한 용도로 활용하실 수 있습니다.

### 카피 의도
- Q1·Q2 → 가장 많이 들어오는 질문으로 구매 장벽 직접 해소
- Q3 → 일정 불안을 숫자로 해소 + "1주일 전 주문 권장"으로 긴급감 생성
- Q4 → 장르 다양성을 나열하여 "내 취향도 되겠다"는 확신 부여
- Q5 → 사용 범위 확장으로 가격 대비 가치(Value) 인식 상승

- **격식체 + 따뜻한 존댓말** ("~합니다", "~드립니다") — 서비스 신뢰감 유지
- **감성적이되 과장하지 않는** 절제된 문장 — 결혼이라는 이벤트의 진정성 존중
- **'AI'라는 단어의 노출 수위** — "AI가 작곡합니다"가 아닌 "최첨단 기술이 제안하고, 전문 디렉터의 귀로 완성합니다" 뉘앙스 유지

### 금지 표현
- ❌ "저렴한", "싼" → ✅ "합리적인", "가성비 있는"
- ❌ "AI가 만든" → ✅ "전문가가 디렉팅한", "프로듀서가 다듬은"
- ❌ "완벽한" (남용 시 신뢰 저하) → ✅ "완성도 높은", "프로 수준의"
